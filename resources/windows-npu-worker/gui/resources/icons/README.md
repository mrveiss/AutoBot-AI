# Application Icons

Place application icons in this directory.

## Required Icons

- `app_icon.png` - Main application icon (256x256 recommended)
- `app_icon_tray.png` - System tray icon (16x16 or 32x32)

## Creating Icons

You can create icons using:
1. **Online Tools**: https://www.favicon-generator.org/
2. **GIMP**: Free image editor
3. **Inkscape**: Vector graphics editor

## Icon Guidelines

- **Format**: PNG with transparency
- **Size**: 256x256 for main icon, 32x32 for tray
- **Style**: Simple, recognizable at small sizes
- **Colors**: Match Windows 11 design language

## Fallback

If no icons are provided, the application uses Qt's default computer icon.
