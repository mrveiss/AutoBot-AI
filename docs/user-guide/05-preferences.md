## Common Questions

### Q: Do my preferences sync across devices?
**A**: Currently, preferences are saved locally in your browser. Each device will have its own preferences. Cloud sync is planned for a future update.

### Q: What happens if I clear my browser data?
**A**: Your preferences will reset to defaults. You'll need to set them again.

### Q: Can I reset all preferences at once?
**A**: Yes! Click the "Reset" button at the top of the Preferences panel to restore all defaults.

### Q: Do preferences affect performance?
**A**: No. Preference changes apply instantly with no impact on performance.

### Q: Are preferences accessible to screen readers?
**A**: Yes! The preferences system is fully accessible with proper ARIA labels, keyboard navigation, and screen reader announcements.

## Accessibility Features

### Keyboard Navigation
- Press **Tab** to move between preference options
- Press **Enter** or **Space** to select an option
- Press **Shift+Tab** to go backwards
- All selections are announced by screen readers

### Screen Reader Support
- All preference changes are announced
- Example: "Font size changed to Large"
- Proper form structure with fieldsets and legends
- ARIA labels on all interactive elements

### High Contrast
- All color combinations meet WCAG 2.1 AA standards

## Troubleshooting

### Problem: Preferences don't save
**Solution**:
- Check that cookies/localStorage are enabled in your browser
- Ensure you're not in incognito/private browsing mode
- Try clearing browser cache and setting preferences again

### Problem: Dark mode doesn't apply
**Solution**:
- Click the dark mode toggle in the header (not in Preferences page)
- Refresh the page
- Check if your browser extensions are interfering

### Problem: Colors look wrong
**Solution**:
- Reset preferences using the "Reset" button
- Try a different browser to rule out browser-specific issues
- Check if your monitor color calibration is correct

### Problem: Text is too small/large after update
**Solution**:
- Go to Preferences
- Select your preferred font size
- If issue persists, reset preferences and try again
