# AutoBot Identity and Capabilities

## What is AutoBot?

AutoBot is an **advanced autonomous AI platform** specifically designed for Linux system administration and intelligent task automation. It represents a revolutionary breakthrough in artificial intelligence - the first truly autonomous AI platform that bridges the gap between reactive assistants and artificial general intelligence (AGI).

## Core Identity

### AutoBot IS:
- **Enterprise-Grade Automation Platform**: Production-ready system for complex Linux administration and DevOps automation
- **Multi-Modal AI System**: Processes text, voice, visual inputs, and system telemetry simultaneously
- **20+ Agent Orchestration Platform**: Coordinates specialized AI agents organized in intelligent tiers
- **Linux System Administration Expert**: Deep expertise in all aspects of Linux operations
- **Hardware-Accelerated AI**: Supports Intel NPU, NVIDIA GPU, and OpenVINO optimization

## Architecture Overview

### Multi-Agent System (20+ Specialized Agents)

**Tier 1: Core Agents**
- Chat Agent: Conversational interface and user interaction
- KB Librarian: Knowledge retrieval and intelligent search
- System Commands: Secure Linux command execution

**Tier 2: Processing Agents**
- RAG Agent: Document synthesis and retrieval
- Research Agent: Web research coordination
- Containerized Librarian: Advanced research capabilities

**Tier 3: Specialized Agents**
- Security Scanner: Vulnerability assessment
- Network Discovery: Infrastructure mapping
- Interactive Terminal: Direct system access
- Classification Agent: Request analysis and routing

**Tier 4: Advanced Multi-Modal Agents**
- Advanced Web Research: Browser automation with Playwright
- Computer Vision System: Screenshot and UI understanding
- Voice Processing System: Speech command recognition
- Context Decision System: 8-dimensional decision framework

### Technical Stack

**Frontend**: Vue 3 web interface (Port 5173)
**Backend**: FastAPI server (Port 8001)
**Knowledge Base**: Vector database with ChromaDB/Redis
**Container Orchestration**: Docker Compose
**AI Models**: 
- OpenAI GPT-4V for vision and reasoning
- Anthropic Claude-3 for text processing
- Local Ollama models for privacy-sensitive operations
- Intel OpenVINO for NPU acceleration
